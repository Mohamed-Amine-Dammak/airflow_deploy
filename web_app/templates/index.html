Core capabilities across releases (current: v0.2)

Visual Pipeline Builder

Drag modules to canvas, connect nodes, pan and zoom comfortably, and keep a clear orchestration flow.

Schema-Driven Configuration

Each module exposes dynamic, conditional fields in the right panel for cleaner and safer configuration.

DAG Generation and Preview

Render readable DAG Python code from templates, inspect output, and deploy to Airflow DAG folders.

Run Monitoring and Logs

Trigger DAG runs, track run/task states live, inspect task logs, and retry failed executions.

Authentication Layer

Protected sessions with login enforcement across pages and APIs, including authenticated logout flow.

Role-Based Access Control

Predefined roles (admin, developer, operator, viewer, connection_manager) gate each feature by permission.

Admin User Management

Admins can create users, assign one or more roles, update role assignments, and remove non-system users.

Custom Role Creation

Create, edit, and delete custom roles with explicit privileges and permission keys from the admin interface.

Airflow Connections CRUD

View, create, edit, and delete supported Airflow connections with role-aware access control.

Pipeline File Workflow

Create new flows, save and load JSON definitions, and manage pipeline state with quick file actions.
